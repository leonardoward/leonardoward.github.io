Interactive Map | OpenLayers

  • February, 2020
  • By Leonardo Ward

This web application contains a world map that runs on the web page. The center of the map is the Arc de Triomphe, its latitude and longitude is marked in the map.

The map requires the Javascript library from OpenLayers, an open source project that facilitates the use of dynamic maps in the web page. It can display map tiles, vector data and markers.

The map is interactive, it's possible to zoom in or out using the + and - buttons or the scrolling wheel on the mouse. It's also possible to change the center position of the map by moving the mouse while being on top of the map and holding left click.
