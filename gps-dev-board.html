GPS Development Board | ANTENOVA M20050-1 and ATGM332D-5N31

  • September, 2021
  • By Leonardo Ward

  • Main Project

  • Github

This project contains the design of a development board for the ANTENOVA M20050-1 and the ATGM332D-5N31. The PCB was designed using KiCad.
